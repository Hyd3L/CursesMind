# CursesMind
A simple ncurses version of the popular MasterMind game.  

## How to install, run, do stuff
```shell
# Clone this repository
git clone https://github.com/hyd3l/cursesmind

# If you don't have ncurses, install it

# Compile the sources
make

# How to install
sudo make install

# Run the game
cursesmind

# Uninstall  
sudo make uninstall
```
